# openGL playground

Just some samples to play around with openGL and getting hands dirty

## Prerequisites

- [just](https://github.com/casey/just)
- [cmake](https://cmake.org/)

## Building

```shell
just build
```

## Running

```shell
just run <bin/example>
```

i.e.
```shell
just run open_gl_starter
just run moving_triangles
```
